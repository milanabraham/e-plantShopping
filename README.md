## Introduction

This repository was created as part of IBM's **"Developing Front End Apps with React"** course.

Paradise Nursery Shopping Cart App is a React-based web application that allows users to browse plants in an online nursery and add them to a shopping cart.

Users can:

- View a list of available plants
- Add items to the shopping cart
- See visual feedback when items are selected
- View cart contents
- Increase or decrease product quantity
- Continue shopping after reviewing cart items

The checkout functionality was not implemented as it was outside the project scope.
